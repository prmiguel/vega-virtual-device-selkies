# kvd-docker

Get the following information from [here](https://developer.amazon.com/docs/vega/0.21/install-vega-sdk.html#ubuntu)
```sh
export SDK_VERSION=
export INSTALLER_SCRIPT=
export SDK_URL=
export SIM_URL=
export DIRECTED_ID=
export KEPLER_TOKEN=
export KEPLER_CID=
export NPM_REGISTRY=
```

```sh
docker compose build
docker compose up -d
docker compose down -v
```
